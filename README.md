## FAQ

### Can I use a normal daily photo?
Yes. AnyIdPhoto is designed to start from a casual photo and convert it to an ID-compliant format.

### Do I get both digital and print outputs?
Yes. You can get digital files and print-ready output (including 4x6 layout options).

### Are all countries using the same passport photo spec?
No. Requirements vary by country and document type. Always verify final specs against the issuing authority.

### Does this repository contain the production core code?
No. This is a public-facing information repository only.
