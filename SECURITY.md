# Security Policy

## Reporting a Vulnerability

If you discover a security issue related to AnyIdPhoto, please report it privately:

- Email: help@anyidphoto.com
- Website: https://anyidphoto.com

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any proof-of-concept details (if available)

We will review reports and respond as quickly as possible.
